# Python提取指定站点属性数据
从原始csv站点数据中提取指定站点的属性变量数据到新csv文件中，格式及提取的属性将参考tem.csv文件，结果将分站点保存该站点所有时间段的属性数据

input_path 为输入路径（文件夹）
output_path 为输出路径（文件夹）
point_names 为想要提取的站点列表
